圖片標註快手 (EZ-PicNote)

版本號：v2.0.0 (旗艦版 - 包含防盜與PWA)

企劃與互動界面設計：吳佩芳

✨ 本APP的特色

  • • 純前端零隱私外洩：無需上傳伺服器，安全處理私密截圖。
  • • 神級智慧工具：自動遞增步驟印章、無縫吸色滴管、去背還原擦。
  • • 直覺圖層管理：支援滑鼠拖曳排序，精準掌握圖層覆蓋關係。
  • • 白板模式支援：無需底圖，直接使用畫筆或印章，瞬間變身教學白板。

📖 操作說明

  • • Win+Shift+S (最強截圖技)：使用電腦內建功能畫框框，然後在APP內直接 Ctrl + V 貼上成為底圖。
  • • 空白畫布：不用貼圖，直接選擇上方畫筆或印章，就能在畫布上自由作畫！
  • • 完美複製物件 (Ctrl+拖曳)：選取任何印章或物件，按住 Ctrl 鍵再用滑鼠拖曳即可完美拉出一個分身！步驟印章還會自動+1號碼！
  • • 遮蔽筆與滴管：使用智慧滴管吸取原圖底色，系統會自動切換為遮蔽筆，讓您無縫隱藏敏感資訊。
  • • 裁切底圖：點選裁切刀拉出範圍，瞬間切除多餘畫面。
  • • Ctrl + 滾輪：直覺放大 / 縮小畫布。
  • • 步驟印章：點擊按鈕一下蓋完單個會自動切換回指標；對按鈕雙擊可鎖定連續蓋章模式。
  • • 快速編輯文字：雙擊文字/印章可修改內容。編輯時按下 Ctrl + Enter 即可快速完成輸入。
  • • 還原橡皮擦：專門擦除您畫錯的標記與畫筆，就像真的白板擦一樣。
  • • 智慧防呆退號：蓋錯印章只要刪除它，號碼會自動退回，不用手動改數字！
